# 📡 NetDiag.py: クロスプラットフォーム対応 ネットワーク診断ツール

このPythonスクリプトは、Windows、Linux、macOS環境の**ネットワーク管理とトラブルシューティング**を支援するための多機能な診断ツールです。

診断タスクの多くは**並列実行**され、結果は自動でログファイルに記録されます。

### 📌 主な機能

* **自動管理者昇格**: Windowsで実行する際、自動的に管理者権限に昇格します。
* **ネットワーク情報収集**: IP設定（`ipconfig /all`など）やリンク速度情報を取得します。
* **接続テスト**: トレースルート（ホップ数計測）および速度テスト（Ping/Download/Upload）を実行します。
* **DNS設定管理 (Windows)**: Windows環境のアダプターに対し、DNSサーバーを自動で設定（または確認）します。

### 💾 実行とログ記録

* 診断結果は、実行日時を含むファイル名（例: `Netcheck2511161240.log`）でスクリプトと同じディレクトリに保存されます。
